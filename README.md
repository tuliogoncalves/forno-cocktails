# THE BAR - Menu de Drinks

## Visão Geral
Um menu digital elegante para bar, exibindo drinks clássicos e tropicais com design moderno em tema escuro com detalhes dourados.

## Estrutura do Projeto
```
/
├── index.html          # Página principal do menu
├── server.py           # Servidor HTTP Python simples
├── static/
│   └── images/         # Imagens dos drinks (stock photos)
└── attached_assets/    # Assets originais e imagens de stock baixadas
```

## Tecnologias
- HTML5 + CSS3 (design responsivo)
- Python 3.11 (servidor HTTP)
- Imagens de stock (Pexels/Unsplash)

## Executando o Projeto
O servidor roda na porta 5000:
```bash
python server.py
```
